#API Demo

天気情報ミニアプリ（Open-Meteo API）

デザイン / コーディング

概要

非同期処理の仕組みを理解するため、API取得を実装しました。

ターゲット

都内在住で外出が多く、朝の短時間で天気を確認したい人

制作期間

WF
2時間
デザイン
2時間
コーディング
1時間

使用ツール

Figma / HTML / CSS(BEM) / Java Script / Fetch API / Open-Meteo API / GitHub

工夫した点

既存の静的コーディング作品に加え、動的処理への理解を深めるため外部API（Open-Meteo）を用いた天気アプリを制作しました。
fetchとasync/awaitによる非同期通信を実装し、地名検索から緯度経度取得、天気データ取得までの流れを設計しています。APIレスポンスから必要なデータのみを抽出し、UIへ反映。エラー時にはメッセージ表示を行うなど、分岐処理も組み込みました。静的なレイアウト制作とは異なり、「データ取得→加工→表示」の一連の流れを意識した構成が本作の工夫です。
公開を前提としたため、APIキーが不要なサービスを選定しました。
